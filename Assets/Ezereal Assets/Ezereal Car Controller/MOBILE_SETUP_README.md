# Настройка мобильного управления для Ezereal Car Controller

## Обзор изменений

Были добавлены следующие компоненты для мобильной игры:

1. **MobileCarController** - контроллер для обработки тач-ввода
2. **Joystick** - виртуальный джойстик для управления рулем
3. **MobileCarUICreator** - утилита для автоматического создания UI элементов
4. **EzerealCameraController** (обновлен) - автоматически переключается в мобильный режим с высоким углом обзора

## Быстрая настройка

### Вариант 1: Автоматическое создание UI

1. Добавьте компонент `MobileCarUICreator` на любой GameObject в сцене
2. Убедитесь, что опция **Enable In Editor** включена (для тестирования в редакторе)
3. Запустите игру - UI элементы создадутся автоматически
4. Или в редакторе: ПКМ на компоненте → "Create Mobile UI"

**Важно для тестирования в редакторе:**
- Джойстик работает с мышью - просто зажмите левую кнопку мыши на джойстике и двигайте
- Кнопки работают при клике мышью
- EventSystem создается автоматически

### Вариант 2: Ручная настройка

1. Создайте Canvas в сцене (если его нет)
2. Добавьте компонент `MobileCarController` на GameObject
3. Назначьте ссылку на `EzerealCarController`
4. Создайте UI элементы:
   - **Джойстик для руля** (левая сторона экрана):
     - Добавьте GameObject с компонентом `Joystick`
     - Настройте background и handle (RectTransform)
   - **Кнопки управления** (правая сторона экрана):
     - Кнопка газа (зеленая)
     - Кнопка тормоза (красная)
     - Кнопка ручного тормоза (оранжевая)
5. Назначьте все элементы в `MobileCarController`

## Настройка камеры

Камера автоматически переключается в мобильный режим на Android/iOS устройствах.

### Параметры мобильной камеры (в EzerealCameraController):

- **Mobile Camera Height** (15) - Высота камеры над машиной
- **Mobile Camera Distance** (20) - Расстояние от машины
- **Mobile Camera Angle** (45) - Угол наклона камеры в градусах
- **Camera Follow Speed** (5) - Скорость следования камеры

### Ручное переключение режима:

```csharp
cameraController.SetMobileMode(true); // Включить мобильный режим
cameraController.SetMobileMode(false); // Выключить мобильный режим
```

## Настройка чувствительности

В компоненте `MobileCarController`:
- **Steering Sensitivity** - Чувствительность руля (по умолчанию 1.0)
- **Use Accelerometer** - Использовать акселерометр вместо джойстика (опционально)

## Примечания

- Мобильный режим камеры автоматически активируется на Android и iOS
- UI элементы создаются автоматически при запуске, если включен `createOnStart`
- Все кнопки используют EventTrigger для обработки нажатий и отпусканий
- Джойстик поддерживает dead zone и настройку чувствительности

## Тестирование в редакторе

Для тестирования мобильного управления в редакторе Unity:
1. Включите мобильный режим камеры вручную: `cameraController.SetMobileMode(true)`
2. Убедитесь, что Canvas настроен на Screen Space - Overlay
3. Используйте симулятор тач-ввода Unity или тестируйте на реальном устройстве
